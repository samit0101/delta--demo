# Mini Project — Counter

This is a tiny static mini-project demonstrating a simple counter UI.

Files:
- `index.html` — main page
- `style.css` — minimal styles
- `script.js` — small JS to increment/decrement/reset the counter

To view locally, open `index.html` in your browser or run a static server:

```sh
cd delta--demo/mini-project
python3 -m http.server 8000
# then open http://localhost:8000
```
